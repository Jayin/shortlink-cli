## shortlink-cli

生成/还原短连接命令行接口

### install

```bash
npm install -g shortlink-cli
```

### Usage

```bash
# 生成短链接
shortlink gen [options] longurl
# 还原短链接
shortlink restore shorturl
```

#### options

* `--baidu` 使用百度短连接服务 http://dwz.cn/
* `--weibo` 使用新浪微博短连接服务 http://open.weibo.com/wiki/2/short_url/shorten
